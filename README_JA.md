## ?? プロジェクト概要

これは原神の機能を大幅に強化し、現代化したFPSアンロッカーです。ゲーム体験を向上させるために設計されました。

**主な特徴：**
- ?? **120FPS** - 60フレームレート制限を完全に解除
- ?? **ネイティブHDRサポート** - HDRレジストリ値を自動配置
- ?? **6言語インターフェース** - 完全な多言語サポート
- ?? **リアルタイムホットキー** - 右Ctrl + 矢印キーでFPS調整
- ?? **自動検出** - 中国版と国際版を自動識別
- ?? **リアルタイムフィードバック** - FPSとHDRステータスを表示
- ?? **設定の永続化** - 設定を自動保存
- ?? **デュアルバージョンサポート** - YuanShen.exeとGenshinImpact.exeに対応

**現在のバージョン:** v10270  
**ライセンス:** Apache License 2.0

## ?? インストール使用法

### ステップ1：ダウンロード
[リリースページ](../../releases)から最新バージョン v10270 をダウンロード

### ステップ2：実行
**管理者権限**でプログラムを実行

### ステップ3：初回設定（初回のみ）
1. 設定ファイルが見つからないというメッセージが表示
2. **ゲームを手動で起動**
3. プログラムが自動的にゲームパスとプロセス情報をキャプチャ
4. ゲームを手動で終了
5. プログラムが自動的に再起動

### ステップ4：通常使用
その後、プログラムを実行するたびにゲームが**自動起動**

## ?? ホットキーリファレンス

ゲーム実行中に使用：

```
右Ctrl + ↑  →  FPS +20
右Ctrl + ↓  →  FPS -20
右Ctrl + →  →  FPS +2
右Ctrl + ←  →  FPS -2
END        →  60 FPSと前の値を切り替え
```

## ?? 技術的特性

### HDR自動設定
- ゲームバージョンを自動検出（原神/国際版）
- 対応するレジストリパスにHDR設定を書き込み
- レジストリの手動編集不要

**設定パス：**
- 原神: `HKEY_CURRENT_USER\Software\miHoYo\原神\WINDOWS_HDR_ON_h3132281285 = 1`
- 国際版: `HKEY_CURRENT_USER\Software\miHoYo\Genshin Impact\WINDOWS_HDR_ON_h3132281285 = 1`

### メモリリアルタイム修正
- ゲームメモリ内のFPS値を直接修正
- シェルコード注入で実装
- ゲーム実行中の動的調整が可能

## ?? よくある質問

### Q1: ゲームが自動起動しない
**A:** `fps_config.ini` のゲームパスが正しいか、ファイルが存在するか確認

### Q2: HDRが表示されない
**A:**
1. モニターがHDRをサポートしているか確認
2. Windows設定でHDRを有効にする
3. ゲーム設定でもHDRを有効にする

### Q3: ホットキーでFPSが変わらない
**A:**
1. ゲームを再起動
2. コンソールウィンドウがバックグラウンドで実行中か確認
3. ホットキーを何度か押してみる
