## English

### ?? About This Project

This is an **enhanced and modernized** FPS unlocker for Genshin Impact, featuring:
- **Native HDR support** for enhanced visual quality
- **Multi-language interface** (5 languages supported)
- **Advanced registry optimization** for game settings
- **Real-time FPS adjustment** via hotkeys
- **Automatic game detection** for both CN and International versions

**Current Version:** v10270  
**License:** Apache License 2.0

### ? Key Features

| Feature | Description |
|---------|-------------|
| ?? **Unlimited FPS** | Remove frame rate caps completely |
| ?? **HDR Support** | Enable native High Dynamic Range mode |
| ?? **Multilingual** | Full support for 6 languages |
| ?? **Hotkey Control** | Adjust FPS in real-time with Right Ctrl + Arrows |
| ?? **Auto-Detection** | Automatically find and launch your game |
| ?? **Live Status** | Display current FPS and HDR status |
| ?? **Config Persistence** | Save and restore your settings |
| ?? **Dual Version Support** | Works with both YuanShen.exe and GenshinImpact.exe |

### ?? Installation

1. Download the latest release from [Releases](../../releases)
2. Extract the executable file
3. Run as Administrator
4. **First run:** The program will guide you to manually launch the game to capture its path
5. **Subsequent runs:** The game will launch automatically

### ?? Hotkey Reference

```
Right Ctrl + ↑  →  Increase FPS by 20
Right Ctrl + ↓  →  Decrease FPS by 20
Right Ctrl + →  →  Increase FPS by 2
Right Ctrl + ←  →  Decrease FPS by 2
END             →  Toggle between 60 FPS and previous value
```

### ?? Technical Details

**Registry Auto-Configuration:**

- No manual registry editing required

**Memory Optimization:**

- Efficient shellcode injection for game process modification
- Real-time synchronization between unlocker and game process

### ?? Troubleshooting

**Q: Game not launching automatically?**  
A: Verify the game path in `fps_config.ini` is correct

**Q: HDR not working?**  
A: Ensure your display supports HDR and it's enabled in Windows Settings

**Q: FPS changes don't apply?**  
A: Try restarting the game or adjusting with hotkeys
